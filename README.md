可以根据后台的数据来进行处理评论框，如果有数据就不允许编辑，如果没有就可以编辑提交，还可以通过ref来限制最少字数（最少150字，最多200字）<br/>
http://yangzhedi.github.io/react/commentTextarea.html<br/><br/>
